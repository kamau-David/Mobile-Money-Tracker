# KES Tracker Backend 🇰🇪

### AI-Powered Financial Intelligence for M-Pesa

KES Tracker is a professional fintech backend built to automate personal finance for the Kenyan market. By leveraging the **Google Gemini API**, it transforms raw M-Pesa SMS data into actionable financial insights, automated budgeting, and goal-oriented savings tracking.

---

## 🚀 Core Features

- **AI SMS Parsing:** Automatically extracts Transaction IDs, amounts, merchants, and balances from raw M-Pesa text.
- **Smart Forecasting:** Uses AI to calculate daily "burn rates" and predict exhaustion dates based on current spending.
- **Savings Guard:** Dynamically calculates "Spendable Balance" by reserving funds for specific goals (e.g., University Fees).
- **Contextual Reminders:** Generates friendly, Kenyan-style bill reminders using Natural Language Processing.
- **Proactive Suggestions:** Analyzes incoming income and suggests contributions toward active savings goals.

---

## 🛠 Tech Stack

- **Runtime:** Node.js (v22+)
- **Framework:** Express.js
- **Database:** PostgreSQL
- **AI Engine:** Google Gemini 1.5 Flash
- **Authentication:** JWT (JSON Web Tokens)
